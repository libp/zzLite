## 值得读读-小程序

### 介绍
* 史上最简单的微信小程序
* 微信小程序实现优质文章阅读

### 已实现功能
* 下拉刷新
* 首次登陆弹窗提示
* 数据储存、提取
* 网络请求
* 转发、分享
* 了解更多页面跳转
* 客服功能

### 安装使用
* 在小程序后台配置：https://www.nichuiniu.cn为请求的域名地址
* 克隆到本地，通过微信小程序开发软件打开即可
